Addendum:

DOCTYPE - Informs the browser that this is an HTML5 document.

html - The primary container element for all HTML code.

body - Contains all visible content displayed to the user.

h1 - Defines a top-level heading.

article - Specifies a section of independent, self-contained content.

h2 - Defines a second-level heading within an article.

p - Creates a paragraph of text.

b - The b element is used to apply bold formatting to text.
